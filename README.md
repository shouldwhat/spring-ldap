# spring-ldap
![](/images/class.png)
-. Ldap(Protocol) 을 제어하여, MS-ActiveDirectory 플랫폼 상의 Entity를 제어하기 위한 프로젝트.

-. OOP 사상을 반영한 클래스 구조 설계.

-. 테스트 가능한 restful api 구현.

-. 제어 가능한 Entity

	(1) 사용자 (CN)
	
	(2) 조직 (OU)
	
	(3) 컴퓨터 (COMPUTER)
	
	(4) 정책 (POLICY)
	
-. 개발 환경

	(1) JDK : 1.8.0_144
	
	(2) spring-boot : 2.0.0.RELEASE
	
	
-. 클래스

	(1) 구조
	
	(2) 설명
